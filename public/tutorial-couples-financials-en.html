üöÄ 2. Getting Started

How to Access the Application

1System Access

Access the application through your web browser at the provided address.

2Create Account or Login

If it's your first access, click "Create Account" and fill in your details. If you already have an account, use "Login".

3Initial Configuration

Set your currency preferences, language, and configure your basic profile.

Main Interface

The interface is divided into main sections:

  • Header: Main navigation, user settings
  • Dashboard: Financial overview
  • Side Menu: Quick access to functionalities
  • Content Area: Display of information and forms

üí° Tip

We recommend starting by configuring your profile and adding your main bank accounts before recording transactions.

üìä 3. Financial Dashboard

Overview

The Dashboard is the application's control center, providing a consolidated view of your finances through informative cards and interactive charts.

Main Cards

üí∞ Total Balance

Sum of all active bank accounts, converted to your preferred currency.

üìà Monthly Income

Total financial inflows for the current period.

üìâ Monthly Expenses

Total expenses incurred in the current period.

üíé Net Balance

Difference between income and expenses (Income - Expenses).

View Modes (For Couples)

1"Both" View

Displays consolidated financial data for the couple.

2Individual View

Allows filtering data by specific user (User 1 or User 2).

Charts and Analysis

  • Pie Chart: Distribution of expenses by category
  • Bar Chart: Monthly evolution of income vs expenses
  • Monthly Comparison: Percentage variation compared to previous month

üí° Usage Tip

Use the "Refresh" button in the top right corner to sync data in real time.

üí≥ 4. Transaction Management

Add New Transaction

1Access Form

In the Dashboard, click the "New Transaction" button or access the "Transactions" tab.

2Transaction Type

Select between "Income" (money inflow) or "Expense" (spending).

3Basic Information

  • Amount: Transaction amount
  • Description: Expense/income details
  • Date: Transaction date
  • Category: Classification (Food, Transportation, etc.)

4Payment Method

Select the bank account or card used in the transaction.

Available Categories

üçî Food

Restaurants, supermarket, delivery

üöó Transportation

üè† Housing

üéØ Entertainment

Managing Existing Transactions

  • View: Complete list with date and category filters
  • Edit: Click the edit icon to modify transactions
  • Delete: Remove incorrect transactions with confirmation
  • Search: Find specific transactions using the search bar
