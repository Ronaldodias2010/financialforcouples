üîß 14. Troubleshooting

Common Problems and Solutions

üîê Login Problems

Forgot my password

Solution: Click "Forgot password" on login screen and follow instructions sent by email.

Email not confirmed

Solution: Check your spam folder and click the confirmation link. If not received, request new email.

üíæ Data Problems

Transaction doesn't appear

Solution: Check if date is in current month and no filters are active. Click "Refresh" on dashboard.

Incorrect balance

Solution: Confirm all transactions were recorded correctly and accounts are active.

üë• Couple Problems

Invitation didn't arrive

Solution: Check if email is correct and ask partner to check spam. Try sending again.

Data doesn't sync

Solution: Both users should logout and login again. Wait a few minutes for synchronization.

Performance Tips

  • Internet: Maintain stable connection for better experience
  • Browser: Use updated versions of Chrome, Firefox, or Safari
  • Cache: Clear browser cache if experiencing slowness
  • JavaScript: Make sure JavaScript is enabled

Technical Support

üìß How to Get Help

  • Email: Contact through support email
  • FAQ: Check frequently asked questions section
  • Chat: Use online chat during business hours
  • Status: Check for scheduled maintenance

Data Security

üîí Security Recommendations

  • Use strong, unique passwords
  • Enable two-factor authentication
  • Don't share login credentials
  • Regularly update your password
  • Access only from trusted devices

Application Updates

The application is automatically updated. New features and improvements are deployed regularly. If you experience issues after an update:

  • Clear your browser cache

  • Check our status page for announcements
